## Deployd 모듈 설정 예제

### 시작하기

1. Deployd 모듈 초기화 명령어

```bash
$ npm run create:db
```

2. Deployd 서버 띄우기

```bash
$ npm run start:server
```

3. Deployd 서버 대시보드와 함께 띄우기
```bash
$ npm run start:dashboard
```

서버가 띄워진 후에는 `localhost:2403` 에서 확인이 가능합니다. 대쉬보드에 접속하려면 `localhost:2403/dashboard` 에 접속하면 대시보드 역시 확인 가능합니다.
